MetaSSH is a powerful SSH connection and management utility.

## Setup

### Prerequisites

- Git must be installed on your system
- Download from: https://git-scm.com/download/win

### Connecting to GitHub Repository

The repository has been created at: **https://github.com/metasina3/MetaSSH**

To connect your local project to GitHub:

1. Install Git if you haven't already
2. Run the setup script:
   ```powershell
   .\setup.ps1
   ```

Or manually:
```powershell
git init
git remote add origin https://github.com/metasina3/MetaSSH.git
git add .
git commit -m "Initial commit: MetaSSH project setup"
git branch -M main
git push -u origin main
```

**Note:** When pushing, use your GitHub username (`metasina3`) and your personal access token as the password.
